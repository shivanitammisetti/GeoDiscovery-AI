# 🌍 GeoDiscovery AI

GeoDiscovery AI is a full-stack web platform designed to help users explore, analyze, and interact with geographical and space-related datasets through a unified and user-friendly interface.

The application integrates structured datasets, dynamic content (blogs, events), and user-driven features (feedback, profiles) into a centralized dashboard, making data exploration intuitive and engaging.

Modern data exploration platforms often lack interactivity and personalization. GeoDiscovery AI addresses this by combining:

* Dataset discovery
* Interactive UI modules
* User-centric features
* Scalable backend APIs

⚙️ Setup Instructions
1. Clone Repository
```
git clone https://github.com/shivanitammisetti/GeoDiscovery-AI.git
cd GeoDiscovery-AI
```
2. Install Backend Dependencies
```
cd backend
npm install
```
3. Configure Environment Variables
Create a .env file inside backend/:
PORT=5000
MONGO_URI=your_mongodb_connection_string
4. Run Backend Server
```
npm start
```
